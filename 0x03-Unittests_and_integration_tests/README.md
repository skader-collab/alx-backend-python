# 0x03-Unittests_and_integration_tests

This directory contains unit and integration tests for the `utils` and `client` modules.

## Files
- `test_utils.py`: Unit tests for `utils.access_nested_map` and related functions.
- `test_client.py`: Unit and integration tests for the `GithubOrgClient` class and related functionality.

## How to Run Tests

```bash
python3 -m unittest discover .
```

## Requirements
- Python 3.x
- `parameterized` package

Install dependencies with:
```bash
pip install parameterized
```
